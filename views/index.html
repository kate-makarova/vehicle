Vehicle Test Task

To get all the data currently stored in the database, go to http://localhost:8080/all.

To upload data into the database, go to http://localhost:8080/download.

To use Apollo GraphQL user interface go to http://localhost:4000.

To schedule update tasks for the future:

  • choose how often you need to update the data; you can set an interval in minutes, hours and days
  • go to http://localhost:8080/schedule/timeUnit/timeNumber
for example, to update the data every 5 minutes go to http://localhost:8080/schedule/minute/5.
